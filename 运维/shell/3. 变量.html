3.1 定义变量的规范

name=ymy
name="ymy"
name='ymy'
  • ""会先解析字符串中是否有引用的变量,''会直接将其中的字符串输出出来
  • =两边不能有空格
  • 如果变量值中没有任何空白符(空格,缩进)等,可以不使用引号引起来

3.1.1 修饰符

readonly:只读变量
使用readonly关键字修饰的变量是只读变量,后续流程中不允许修改

unset:删除变量
使用unset关键字修饰的变量标识该变量被删除,后续引用不会生效

3.2 使用变量

3.2.1 调用变量

在变量名前面加上$进行调用即可,如果不是很好区分边界值,则可以在${}变量两边加上{}进行区分

skill="Java"
echo “I like ”$skill
echo "I am good at ${skill}Script"

3.2.2 字符串操作

拼接字符串

直接使用${}包裹起来

name="jetty"
echo "my name is ${name}!"
echo "my name is "$name"!"

获取字符串长度

使用${#}

name="jetty"
echo "my name size is ${#jetty}"

print 4

提取子字符串

使用${string:1:4}


name="jetty"
echo "my name is ${string:1:4}"

print etty

查找子字符串

使用 expr index "$string" io

在string变量中找io字符串出现的位置.找不到则获取其中一个字符的位置
