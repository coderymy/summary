1. 安装vue的脚手架工具（官方命令行工具）

   npm install --global vue-cli

2. 创建项目

   vue init webpack vue-demo1

3. 安装依赖

   cd vue-demo1

   cnpm install

4. 运行项目

   npm run dev

还有一种创建项目的方式

vue init webpack-simple vue-demo1

这种创建方法创建的项目比较简单，这么目录结构比较简单
