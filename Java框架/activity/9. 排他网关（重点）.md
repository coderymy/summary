# 流程图构建

构建一个费用在[500-]，[500~1000]，[100+]的排他网关
1. 构建如img中的排他网关的图
2. 给每条线取一个名字
3. 给每条线的Main config的Condition创建el表达式
    1. ${money>0&&money<=500}
    2. ${money>500&&money<=1000}
    3. ${money>1000}
4. 在排他网关的设置中有一个Default flow，可以选择一个作为默认的情况(作为默认的情况就不需要再加条件了)
