# 并行概念
比如，网上购物。物流和入账，是一个并行的网关
<br/>
也就是说，我买东西，商品在送往客户途中，这个价钱也在送往商家账户的途中，只有当客户确认收货了，钱才能同时到达商家的账户

也就是两条流程是同时执行的，并不是向之前一样需要满足某个情况来执行其中的一条
### 分支
并行后的所有外出顺序流，为每个顺序流都创建一个并行分支
### 汇聚
分支到达并行网关之后，需要等待所有的并行分支都到达并行网关之后才开始信息下面的任务或者完成流程
